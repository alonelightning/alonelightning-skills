# 环节六：ICP 备案（纯人工流程 + agent 陪跑指引）

⚠️ **自动化现状（已核实）**：阿里云备案系统没有对外 OpenAPI，且流程含证件上传、阿里云 App 人脸核验、工信部短信核验、管局审核——**全程无法由 agent 代办**。agent 的角色是：判断要不要备案、备材料清单、每一步给准确入口和填写要点、盯时间节点提醒用户。政策随管局调整，最终以阿里云备案系统 https://beian.aliyun.com 当前要求为准。

> 播报模板（进入本环节）：
> 「备案是政府审核流程，必须你本人操作（要人脸识别），我没法代办，但我会告诉你每一步在哪、填什么、怎么不被驳回，全程陪跑。顺利的话 1~3 周下来。」

## 1. 要不要备案？（agent 先判断）

- 域名解析到**中国内地**服务器对外提供服务 → **必须备案**，否则被阻断。
- 中国香港/海外服务器 → 无需 ICP 备案。
- 与端口/用途无关：API 接口、非 80 端口，只要域名指向内地 IP 就要备。
- 只用 IP 不绑域名 → 不涉及备案（但无法正经上线）。
- 在哪家服务器就在哪家备：用阿里云服务器就走阿里云备案系统；域名在别处买的也可以（域名需已实名且与备案主体一致）。

## 2. 备案前检查（agent 可帮核对）

| 准备项 | 要求 | agent 能做什么 |
|--------|------|---------------|
| 阿里云账号 | 已实名；一个账号对应一个备案主体 | 问询确认 |
| 服务器 | 阿里云内地 ECS/轻量，包月且剩余时长通常 ≥3 个月 | 用 DescribeInstances 查到期时间核对 |
| 域名 | 已注册+已实名，持有人与备案主体一致 | 提醒核对信息模板 |
| 手机 | 备案负责人本人手机（短信核验用） | 提醒 |

**个人材料**：身份证、本人手机号、居住地址；部分省份要辅助说明/承诺书。
**企业材料**：营业执照、法人身份证、网站负责人身份证（非法人另需授权书）、单位地址与电话；部分管局有额外要求（如法人手机号归属地），以各省规则为准。

**特殊行业前置审批**：新闻、出版、教育、医疗、金融、宗教等需先取得主管部门文件；普通博客/企业官网不涉及。

**内容红线（常见驳回原因）**：
- 个人备案不能有经营性内容（卖货、广告联盟），网站名不能含企业/行业/地名等敏感词。
- 备案名称与实际内容相符；空壳站/待建设页部分管局不过。

## 3. 备案五步（用户操作，agent 陪跑）

入口：https://beian.aliyun.com （或控制台搜「ICP 备案」）。人脸核验在**阿里云 App**（手机端）完成。

1. **填写订单**：基础信息校验（系统自动判断首次备案/新增网站/新增接入）→ 主办者信息 → 网站信息 → 上传证件 → App 人脸核验 → 提交。agent 可逐字段指导填写。
2. **阿里云初审**：1~2 个工作日，审核专员电话/短信联系补正，保持手机畅通；打回就改完重交。
3. **工信部短信核验**：收到 12381 短信后 **24 小时内**按指引完成验证码核验。**超时订单退回——最容易疏忽的一步**，agent 应主动提醒用户盯短信。
4. **管局审核**：法定 20 个工作日内，多数省 1~2 周。结果短信+邮件通知。
5. **备案后处理**：
   - 拿到备案号（如：X ICP 备 XXXXXXXX 号）。
   - **网站页脚放备案号**并链接 https://beian.miit.gov.cn （工信部要求，会核查）——这一步 agent 可自动改页面（环节四通道）。
   - **公安联网备案**：开通后 30 天内到 https://beian.gov.cn 提交，完成后公安备案号也放页脚。
   - 经营性网站另需 ICP 经营许可证。

## 4. 时间线与节奏

- 顺利全程 **1~3 周**；被驳回一次顺延，所以资料一次填对最省时间。
- 最优节奏：**买完服务器当天提交备案**，审核期间用 IP 调试部署（环节四），通过当天解析域名（环节五）上线。
- agent 可设提醒：提交后第 2 天问初审结果、提醒盯 12381 短信、每周问一次管局进度。

## 5. 避坑清单

- **审核期间网站保持不可通过域名访问**：解析已生效且能打开可能被驳回。
- 三对齐：**账号实名 = 域名实名 = 备案主体**。
- 接听 95187（阿里云回访）和管局电话，回答与资料一致。
- 信息变更/注销走正规流程，失实会被工信部抽查注销。
- 从其他云迁来走「新增接入」，原备案号保留，网站不停。
- **备案免费**。一切收费"加急/包过"都是骗局。
