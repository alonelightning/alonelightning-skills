# 环节一：需求分析与选型

目标：通过对话确定「产品线 + 地域 + 实例规格 + 镜像 + 带宽 + 存储 + 计费」，输出一份用户点头认可的选型结论。本环节全部在对话中完成，无外部操作，**唯一可选的自动动作是询价**（见环节三）。

## 0. 必问三件事（信息不全不给方案）

1. **用途**：博客/官网/小程序后端/电商/AI 应用/学习测试？
2. **预算**：一年愿意花多少钱？
3. **访问者在哪**：中国内地还是海外？（直接决定地域和是否要备案）

追问话术示例（对小白）：
> 「先问你三个问题就能定方案：① 这台服务器打算拿来干嘛？② 一年预算大概多少？③ 访问你网站的人主要在国内还是国外？」

## 1. 产品线：轻量应用服务器 vs 云服务器 ECS

| 对比项 | 轻量应用服务器 | 云服务器 ECS |
|--------|--------------|--------------|
| 定位 | 入门级，开箱即用 | 主流企业级，功能全 |
| 上手难度 | 低（控制台简化，预装应用镜像如 WordPress） | 中 |
| 计费 | 套餐制，含固定流量包 | 灵活（带宽单独选） |
| 扩展性 | 有限 | 强（随时升配、挂盘、负载均衡） |
| 适合 | 个人博客、作品展示、学习测试 | 小程序/App 后端、企业官网、数据库、要成长空间的业务 |

**决策口径：**
- 纯个人玩、预算 100 元/年级别 → 轻量应用服务器
- 业务会成长、要接数据库/多服务、小程序后端 → 直接 ECS
- 拿不准 → ECS 经济型或通用算力型，通用性最好

> 注：本 skill 的自动购买/部署路径以 **ECS** 为主（API 最全）。轻量应用服务器也有 OpenAPI（SWAS-OPEN），但活动套餐通常要控制台买，走手动指引即可。

## 2. 地域（Region）——最重要且买后不可改

- **访问者在中国内地** → 内地地域：华北2（北京 cn-beijing）、华东1（杭州 cn-hangzhou）、华东2（上海 cn-shanghai）、华南1（深圳 cn-shenzhen）。选离主要用户近的即可。**内地服务器绑域名必须 ICP 备案**。
- **不想备案 / 面向海外** → 中国香港（cn-hongkong）、新加坡等，免备案，内地访问延迟略高。
- ⚠️ 地域买定不能换，跨地域迁移要重买+迁数据。下单前必须跟用户确认这一点。
- 可用区（A/B/C）新手无所谓，默认即可。

## 3. 实例规格（CPU/内存）

入门两个规格族：

| 规格族 | 特点 | 适合 |
|--------|------|------|
| 经济型 e 实例（如 ecs.e-c1m1.large = 2核2G，ecs.e-c1m2.large = 2核4G） | 共享型 CPU，最便宜 | 个人网站、博客、开发测试 |
| 通用算力型 u1 实例（如 ecs.u1-c1m2.large = 2核4G） | 独享算力，性能稳定 | 企业应用、小程序后端、生产环境 |

按用途建议：

| 用途 | 建议配置 |
|------|---------|
| 个人博客/静态站 | 2核2G（e 实例或轻量） |
| WordPress 等动态博客 | 2核2G 起，带数据库建议 2核4G |
| 小程序/App 后端 | 2核4G（u1） |
| 企业官网（展示型） | 2核2G~2核4G |
| AI 应用（调大模型 API） | 2核2G~2核4G 普通实例即可 |
| 自部署开源模型推理 | GPU 实例（gn 系列），价格高一个数量级，先确认真的需要 |

⚠️ 避坑：1核1G 装个数据库就内存不足，不建议；突发性能 t 系列有 CPU 基线限制，生产慎选。

## 4. 镜像（操作系统）

- 首选 **Ubuntu 最新 LTS**（教程最多）或 Alibaba Cloud Linux（阿里维护、免费）。
- CentOS 7/8 已停维，别选。Windows Server 仅 .NET Framework 老应用需要，内存至少 4G。
- 镜像 ID 会随版本更新，自动下单前用 `aliyun ecs DescribeImages --RegionId <region> --OSType linux --ImageOwnerAlias system` 查当前可用镜像 ID，不要硬编码。

## 5. 带宽

- **固定带宽**：3M/5M 包月买断，流量不限。流量稳定的网站首选，新手最省心。
- **按流量**：峰值可设高，按 GB 计费（约 0.8 元/GB，以官网为准）。⚠️ 被刷流量会产生高额账单，务必设费用预警。
- 3M ≈ 日均几百~上千 PV 的普通网站；图多视频多建议 5M 或上 CDN。

## 6. 存储

- 系统盘 ESSD（Entry/PL0）40G 起步够用。数据多加挂数据盘。
- 云盘 ≠ 备份：建议开快照策略（少量费用，属花钱项 → 需同意）。

## 7. 输出选型结论模板

```
推荐方案：
- 产品：云服务器 ECS（经济型 e 实例）
- 地域：华东1（杭州）cn-hangzhou
- 规格：ecs.e-c1m1.large（2核2G）
- 镜像：Ubuntu 24.04 LTS（下单前查实际镜像 ID）
- 带宽：3M 固定带宽
- 系统盘：ESSD 40GB
- 计费：包年（1年）
理由：……
注意：内地地域需备案；接下来我可以帮你查这套配置的实际价格。
```

结论给出后：
1. 问用户是否认可方案；
2. 认可 → 进入环节二（账号准备）；若账号已就绪 → 环节三询价。
